PHASE C — PDF STRUCTURE

The PDF must be scope-first, not drawing-first.

Required structure:

SECTION 1 — Cover / Project Summary
Include:
- Project / estimate name
- Site address
- Client / builder name if available and appropriate
- Date issued
- Prepared by
- Item count
- Scope type badge: Renovation or New Build
- Work package badge: Install only or Removal + disposal + install
- Statement such as:
  “Issued for subcontractor installation pricing only”
- Site photo(s) if enabled and available

SECTION 2 — Scope Summary
Provide a professional subcontractor-facing summary with headings like:
- Scope Requested
- Scope Includes
- Scope Excludes
- Assumptions
- Potential Variations / To Be Confirmed

For Renovation mode, include truthful content for:
- removal of existing units
- disposal of removed units
- installation of new joinery
- standard pack, level, fix, and seal
- standard site clean-up of installation debris if included
- site works assumptions
- renovation-specific exclusions and variations

For New Build mode, include truthful content for:
- installation of new joinery only
- standard pack, level, fix, and seal to prepared openings
- site works assumptions
- new-build-specific exclusions and variations

This section must read like a professional subcontractor scope document, not internal notes or customer quote text.

SECTION 3 — Scope Detail Matrix
Add a compact project-specific scope matrix showing the selected issue settings, for example:
- Scope type
- Work package
- Sealant
- Flashings
- WANZ bars / support systems
- Site clean-up
- Making good
- Access condition

SECTION 4 — Item Schedule Summary
Include a clean table with columns such as:
- Item ID
- Location / room
- Category
- Layout / handing
- Dimensions
- Qty
- Install notes (if relevant)
- Drawing page reference (if detailed drawings included)

This schedule should be more useful than the current raw all-items PDF.

SECTION 5 — Detailed Item Pages
If enabled, include one or more pages with:
- Item ID
- Qty
- Drawing/image
- Dimensions
- Layout / handing
- Category
- Location / room if available
- Install-relevant notes if available

SECTION 6 — Site / Access / Variation Notes
If enabled or if data exists, include a section covering site pricing risks and practical context such as:
- access restrictions
- upper-level / scaffold implications
- occupied dwelling considerations
- staging / delivery / protection notes
- notes requiring confirmation before final pricing

SECTION 7 — Pricing Return
If enabled, include a subcontractor pricing return section with:

A. Overall pricing block
- Total price for scope: __________________
- plus GST / incl GST: __________________
- Lead time / availability: __________________
- Contact name / company / phone / email

B. Assumptions / exclusions / variation notes
- Assumptions / clarifications
- Exclusions
- Variation risks / items requiring confirmation

C. Optional item-by-item pricing table
Include rows for each item with columns like:
- Item ID
- Location
- Description
- Qty
- Subcontractor price
- Notes

This section should be suitable for printing or digital markup.

PHASE E — EXCLUSIONS / VARIATION GUIDANCE

The generated document must clearly cover typical subcontractor pricing-risk areas, such as:
- structural alterations excluded unless noted
- builder’s making good excluded unless noted
- plastering / painting / tiling / floor repair excluded unless noted
- electrical / blinds / alarms / special hardware removal and reinstatement excluded unless noted
- unexpected framing/opening defects as variation items
- access restrictions / scaffolding / lifting requirements as variation items unless noted
- hidden damage / rot / out-of-square openings as variation items
- sealing / flashing assumptions where relevant
- support bars / WANZ bars where relevant
- travel / staged install / after-hours if relevant

Variation checklist options should cover likely pricing risks such as:
- difficult access
- height over 2m / scaffold
- structural or framing alterations
- hidden damage / rot
- hazardous materials / asbestos
- complex flashing requirements
- support bars / WANZ bars
- staging / travel / urgency

Keep the wording professional and practical.
