OBJECTIVE:
Implement full Xero OAuth 2.0 Authorization Code flow with callback handling and token storage.

---------------------------------------

PART 1 — BACKEND ROUTES

Create the following endpoints:

1. GET /api/xero/connect
- Redirect user to Xero authorization URL
- Include:
  - client_id
  - redirect_uri
  - response_type=code
  - scope:
    offline_access accounting.transactions accounting.contacts
  - state (random string for security)

2. GET /api/xero/callback
- Receive:
  - code
  - state
- Exchange code for:
  - access_token
  - refresh_token
  - expires_in
- Store token set in database (see below)
- Return success response

---------------------------------------

PART 2 — DATABASE (MINIMAL)

Create table:

xero_connections
- id
- tenantId (nullable for now)
- accessToken
- refreshToken
- expiresAt (timestamp)
- createdAt
- updatedAt

DO NOT modify existing invoice tables

PART 3 — TOKEN MANAGEMENT

Implement helper:

getValidXeroToken()
