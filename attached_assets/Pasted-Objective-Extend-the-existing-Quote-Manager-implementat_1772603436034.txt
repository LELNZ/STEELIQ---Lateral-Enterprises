Objective

Extend the existing Quote Manager implementation so that the system supports the following architecture:

Layer 1 — Estimates

Estimator workspaces where items are created.

Example:

LJ – Estimates (Lateral Joinery estimator)

This layer captures:

configuration

dimensions

item image (drawing)

library references

cost/pricing logic

Layer 2 — Unified Quote System

The Quotes module acts as a commercial container that manages:

quote numbers

revisions

audit trail

lifecycle (Draft → Review → Sent → Accepted / Declined → Archived)

customer quote rendering

PDF generation

future Xero integration

future procurement triggers

Quotes are generated from estimates via an immutable snapshot.

Critical Business Rule

Legal entity:

Lateral Engineering Limited

Divisions:

Lateral Joinery (LJ)

Lateral Engineering (LE)

Lateral Laser Cutting (LL)

Quotes from divisions must display:

“A trading division of Lateral Engineering Limited”

Branding is determined by the division.
