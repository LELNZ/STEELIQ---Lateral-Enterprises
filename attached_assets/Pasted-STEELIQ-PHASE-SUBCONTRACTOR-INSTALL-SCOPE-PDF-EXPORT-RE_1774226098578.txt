CURRENT NEED

In the estimate phase, the existing “All Items (PDF)” export is useful as a raw drawing pack, but it is not a proper subcontractor issue document for installation pricing.

The business need is to generate a dedicated PDF that can be sent to subcontractors so they can price installation work.

This export must support two scope modes:

1. Renovation
Includes:
- removal of old units
- disposal of old units
- installation of new units
- a clear explanation of what install includes
- site works / assumptions
- potential variation points
- other considerations relevant to renovation work

2. New Build
Includes:
- installation of new units only
- a clear explanation of what install includes
- assumptions about prepared openings / readiness
- site works / assumptions
- potential variation points
- other considerations relevant to new build work

PRIMARY OBJECTIVE

Create a dedicated subcontractor-facing PDF export from the estimate phase called something like:

- Subcontractor Install Scope PDF
or equivalent truthful naming

This PDF must:
- exclude customer pricing / commercial quote terms
- include a professional item schedule
- include item images/drawings
- include site photo(s) where available
- include a scope summary tailored to Renovation or New Build
- include exclusions / assumptions / variation points
- include a pricing return section the subcontractor can fill in

PHASE C — PDF STRUCTURE

The PDF should be scope-first, not drawing-first.

Required structure:

SECTION 1 — Cover / Project Summary
Include:
- Project / estimate name
- Site address
- Client / builder name if available and appropriate
- Date issued
- Prepared by
- Item count
- Scope type badge: Renovation or New Build
- Statement such as:
  “Issued for subcontractor installation pricing only”
- Site photo(s) if enabled and available

SECTION 2 — Scope Summary
For Renovation mode, include a clean summary of:
- removal of existing units
- disposal of removed units
- installation of new joinery
- what install includes
- site works / assumptions
- exclusions
- potential variation points

For New Build mode, include a clean summary of:
- installation of new joinery only
- what install includes
- assumptions about prepared openings
- exclusions
- potential variation points

This section must read like a professional subcontractor scope document, not internal notes.

SECTION 3 — Item Schedule Summary
Include a clean table with columns such as:
- Item ID
- Location / room
- Category
- Layout / handing
- Dimensions
- Qty
- Notes (if relevant)
- Drawing page reference (if detailed drawings included)

SECTION 4 — Detailed Item Pages
If enabled, include one or more pages with:
- Item ID
- Qty
- Drawing/image
- Dimensions
- Layout / handing
- Category
- Location / room if available
- Install-relevant notes if available

SECTION 5 — Pricing Return
If enabled, include a subcontractor pricing return section with:

A. Overall pricing block
- Total price for scope: __________________
- plus GST / incl GST: __________________
- Lead time / availability: __________________
- Contact name / company / phone / email

B. Assumptions / exclusions / variation notes
- Assumptions / clarifications
- Exclusions
- Variation risks / items requiring confirmation

C. Optional item-by-item pricing table
Include rows for each item with columns like:
- Item ID
- Location
- Description
- Qty
- Subcontractor price
- Notes

This section should be suitable for printing or digital markup.

PHASE D — SCOPE WORDING

Provide professional default wording that varies by scope type.

RENOVATION MODE should include truthful headings/content like:
- Scope Includes
- Removal of existing units
- Disposal of removed units
- Installation of new units
- Standard pack, level, fix, and seal
- Standard site cleanup of install debris
- Assumptions
- Exclusions
- Potential variations

NEW BUILD MODE should include truthful headings/content like:
- Scope Includes
- Installation of new units only
- Standard pack, level, fix, and seal to prepared openings
- Assumptions
- Exclusions
- Potential variations

Do not include customer-facing commercial quote wording or internal margin language.

PHASE E — EXCLUSIONS / VARIATION GUIDANCE

The generated document should clearly cover typical pricing-risk areas, such as:
- structural alterations excluded unless noted
- builder’s making good excluded unless noted
- plastering / painting / tiling / floor repair excluded unless noted
- electrical / blinds / alarms / special hardware removal and reinstatement excluded unless noted
- unexpected framing/opening defects as variation items
- access restrictions / scaffolding / lifting requirements as variation items unless noted
- hidden damage / rot / out-of-square openings as variation items
- sealing / flashing assumptions where relevant

Keep the wording professional and practical.
