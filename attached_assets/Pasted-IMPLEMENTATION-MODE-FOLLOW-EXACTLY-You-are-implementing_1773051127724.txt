T083 — Enforce strict A4 page geometry
--------------------------------------------------

The quote document must be designed for **A4 portrait only**.

PDF engine requirements:

Use explicit jsPDF configuration:

new jsPDF({
  orientation: "portrait",
  unit: "mm",
  format: "a4"
})

Define page constants at the top of the PDF engine:

PAGE_WIDTH = 210
PAGE_HEIGHT = 297

LEFT_MARGIN = 15
RIGHT_MARGIN = 15
TOP_MARGIN = 18
BOTTOM_MARGIN = 18

CONTENT_WIDTH = PAGE_WIDTH - LEFT_MARGIN - RIGHT_MARGIN

All rendering logic must use CONTENT_WIDTH instead of arbitrary widths.

No layout element may exceed this width.

Acceptance criteria:

PDF layout is stable and always fits inside A4 margins.
