CURRENT LIVE ISSUE

In the live estimator, when creating an entrance door with a sidelight, the user currently has to manually calculate and enter the sidelight width.

This is the wrong workflow.

Desired real-world workflow:
1. enter total overall width and height
2. choose sidelight position (left or right)
3. enter main door width
4. system auto-calculates the remaining sidelight width

Example:
- overall width = 1470
- overall height = 2000
- sidelight position = left
- main door width = 800
- calculated sidelight width = 670
