T083 — Enforce strict A4 layout in PDF and Preview
--------------------------------------------------

Problem:
The quote must behave like a true A4 customer document.
Preview and PDF must be based on the same page geometry.

Required implementation:

1. In the PDF engine, explicitly enforce:
   - portrait orientation
   - unit = mm
   - format = a4

2. Define fixed page constants and use them consistently:
   - PAGE_WIDTH = 210
   - PAGE_HEIGHT = 297
   - LEFT_MARGIN = 15
   - RIGHT_MARGIN = 15
   - TOP_MARGIN = 18
   - BOTTOM_MARGIN = 18
   - CONTENT_WIDTH = PAGE_WIDTH - LEFT_MARGIN - RIGHT_MARGIN

3. Remove/replace any arbitrary widths so rendering uses the A4 content width.

4. In Preview, constrain the document container to A4-equivalent width.
   The preview should behave like a document page, not a full-width responsive web page.

Acceptance criteria:
- PDF is explicitly A4 portrait
- Preview width is clearly aligned to A4
- layout tuning becomes predictable between Preview and PDF
